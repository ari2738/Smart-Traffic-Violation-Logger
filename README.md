# 🚦 Smart Traffic Violation Logger

A Flask-based web application designed to log, track, and manage traffic violations digitally. Generates unique QR codes for each violation and provides a public status page for verification and payment tracking.

## ✨ Features

- **Add New Violations**: Log vehicle number, violation type, location, fine amount, and date
- **QR Code Generation**: Auto-generates unique QR for each violation linking to status page
- **Public Status Page**: Anyone can scan QR to check if fine is Paid/Unpaid
- **Admin Dashboard**: View all violations, update payment status, delete records
- **Search & Filter**: Find violations by vehicle number, date, or status
- **Responsive UI**: Mobile-friendly interface using Bootstrap

## 🛠️ Tech Stack

- **Backend**: Python, Flask, Flask-SQLAlchemy
- **Frontend**: HTML, Jinja2, Bootstrap 5
- **Database**: SQLite
- **Other**: qrcode, datetime
